HTML - lenguaje de hipertexto

Estructura Básica

Documentos HTML são divididos em algumas partes básicas. Antes de iniciar os elementos, devemos utilizar uma declaração <!DOCTYPE html> na primeira linha.

O elemento raíz "html" é de onde prtem todos os outros elementos, ele é composto das partes "head" e "body"

Tags y elementos

Na marcação HTML, cada elemento é representado por uma tag de abertura, conteúdo e uma tag de fechamento. tags de abertura são escritas no formato < nome-da-tag> e tags de fechamento no formato </nome-da-tag>

Elementos de texto

Alguns elementos de texto que podemos utilizar:

  • Parágrafo - tag <p>
  • Títulos - tags <h1> a <h6>
  • Links - tag <a>
